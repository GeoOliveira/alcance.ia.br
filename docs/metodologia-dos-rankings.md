# Metodologia dos rankings

## Hashtags

Hashtags são normalizadas em minúsculas e deduplicadas dentro de cada post. O relatório calcula total, média, mediana, posts sem hashtag, exclusivas, repetidas, concentração das cinco principais e desempenho por hashtag quando a amostra mínima configurada é atingida. Associação não implica causalidade.

## Reels

- visualizações: `playCount`, com fallback para `viewCount`;
- engajamento: `(curtidas + comentários) / visualizações`;
- proporcional: `visualizações / seguidores atuais`;
- interações: `curtidas + comentários`.

Campos ausentes excluem o item do critério correspondente. Os rankings representam somente a amostra pública analisada, não todo o Instagram.
