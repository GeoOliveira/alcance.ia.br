# Métricas dos links

A estratégia é consulta sob demanda com cache curto em `user_link_metric_cache`. Somente agregados do Encurta.io podem preencher cliques, evolução diária, dispositivos, referências, humanos estimados e bots.

Como o contrato de consulta não existe no material local, a página não inventa endpoint, não busca eventos individuais e não apresenta demonstrações como dados reais.
