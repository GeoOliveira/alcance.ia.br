# Privacidade do gerenciador

Dados tratados: perfil básico, provedor, consentimentos, telefone, mensagem, metadados dos links, QR Codes determinísticos, preferências e métricas agregadas. Telefone e mensagem são transmitidos ao Encurta.io para o redirecionamento.

Analytics e logs não podem conter telefone, mensagem, e-mail, slug, URL curta, ID token, access token ou segredos. Direitos, retenção e exclusão exigem revisão jurídica antes da ativação.
